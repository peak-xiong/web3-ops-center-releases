# Web3 Ops Center - Downloads

This repository hosts the release binaries for [Web3 Ops Center](https://github.com/peak-xiong/web3-ops-center).

## Installation

### Windows
1. Download the `.exe` installer from [Releases](https://github.com/peak-xiong/web3-ops-center-releases/releases)
2. Run the installer and follow the prompts
3. Launch "Web3 Ops Center" from the Start Menu

### macOS
1. Download the `.dmg` file (choose `arm64` for Apple Silicon, `x64` for Intel)
2. Open the DMG and drag the app to Applications
3. On first launch, right-click and select "Open" to bypass Gatekeeper

### Linux
- **AppImage**: Download, make executable with `chmod +x`, and run
- **Deb**: Install with `sudo dpkg -i <filename>.deb`
